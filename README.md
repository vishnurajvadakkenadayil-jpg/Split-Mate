# Split Mate – Your Ultimate Expense Sharing Companion

Split Mate is a smart, simple, and user-friendly app designed to make splitting bills and tracking shared expenses effortless. Whether you’re traveling with friends, living with roommates, or managing group events, Split Mate keeps everyone on the same page.

Key Features:

Instant Bill Splitting: Add expenses and let Split Mate calculate each person’s share automatically.
Group Management: Create groups for trips, roommates, or events and track who owes what.
Real-time Updates: Everyone in the group sees updates instantly, reducing confusion and arguments.
Payment Tracking: Mark payments as settled and keep a clean record of past transactions.
User-Friendly Interface: Simple, intuitive design so anyone can use it without hassle.

Take the stress out of shared expenses — Split Mate ensures fairness, transparency, and convenience. Never worry about “who owes what” again!
